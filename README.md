# Abstract repository

Библиотека для работы с репозиториями, сервисами и кэшированием данных.

## Документация

### Архитектура

- **[Обзор архитектуры](docs/architecture.md)** - Общее описание архитектуры проекта

### Компоненты

1. [Services](docs/services/README.md) - Сервисы для бизнес-логики
2. [Repositories](docs/repositories/README.md) - Репозитории для доступа к данным
3. [Filters](docs/filters/README.md) - Фильтры для выборки данных
4. [Structures](docs/structures/README.md) - Структуры данных для организации информации
5. [Drivers](docs/drivers/README.md) - Драйверы кэша для работы с хранилищами
6. [Traits](docs/traits/README.md) - Трейты для переиспользуемой функциональности
7. [Exceptions](docs/exceptions/README.md) - Исключения для обработки ошибок
8. [Utils](docs/utils/README.md) - Утилиты для общих функций
9. [Interfaces](docs/interfaces/README.md) - Интерфейсы для определения контрактов
